React Doc Viewer - Sample Text Document
========================================

This is a sample plain text file used to demonstrate the TXT renderer
in @iamjariwala/react-doc-viewer.

Features:
- Lightweight document viewer component for React
- Supports 20+ file formats
- PDF viewing with annotations, thumbnails, and search
- Drag and drop file loading
- Dark mode support
- Internationalization (i18n)

All renderers are tree-shakeable and can be imported individually.
